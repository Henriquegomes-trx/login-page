# 🔐 Login Page

Uma tela de login moderna desenvolvida para praticar HTML5 e CSS3, com foco em layout responsivo, efeitos visuais e boas práticas de estruturação de código.

## 📸 Preview

> Adicione aqui uma imagem da aplicação.

```md
![Preview](preview.png)
```

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Interface moderna
- Layout responsivo
- Efeito Glassmorphism
- Animações suaves com `transition`
- Efeitos `hover` em botões e links
- Organização semântica do HTML
- Design limpo e intuitivo

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/Henri-quemes-trx/login-page.git
```

2. Entre na pasta do projeto.

3. Abra o arquivo `index.html` no navegador.

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento Front-end, incluindo:

- Estruturação semântica com HTML5
- Estilização utilizando CSS3
- Flexbox
- Responsividade
- Animações e transições
- Organização de código

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

- Posicionamento com Flexbox
- Pseudo-classes (`:hover`)
- Transições (`transition`)
- Transformações (`transform`)
- Glassmorphism
- Organização de arquivos
- Versionamento com Git e GitHub
