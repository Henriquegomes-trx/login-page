# Login Page

Tela de login desenvolvida com HTML e CSS para praticar conceitos de Front-end, como Flexbox, responsividade e efeitos visuais.

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprimorar conhecimentos em HTML e CSS, criando uma interface moderna e organizada.

### Funcionalidades

- Tela de login responsiva
- Efeitos `hover` em botões e links
- Layout moderno
- Design Glassmorphism
- Estrutura semântica em HTML

## Tecnologias utilizadas

- HTML5
- CSS3
- Git
- GitHub
